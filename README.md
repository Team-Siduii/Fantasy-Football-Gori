# Fantasy-Football-Gori

Basis skeleton voor de nieuwe app.

## Stack
- Next.js (App Router)
- TypeScript (strict)
- ESLint

## Lokale start
1. npm install
2. npm run dev
3. open http://localhost:3000

## Deployment
- Vercel project is gekoppeld.
- Elke push naar een branch kan als preview deploy draaien.
- `main` is productie-branch.

## Volgende stappen
- UX/design uitwerken
- domeinmodel (leagues, teams, drafts, picks)
- authenticatie + rollen
- data-opslag keuzes
